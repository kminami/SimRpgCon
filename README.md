# SimRpgCon

## これは何？

SRC (Simulation RPG Construction) をブラウザに移植しようという試みです。

## 使い方

SRCのフォルダ（srcall-2_2_33-111106など）の直下にSimRpgCon.htmlを配置して、ブラウザで実行して下さい。

### 配置例
```
srcall-2_2_33-111106
 |- Bitmap / ...
 |- Data / ...
 |- Sound / ...
 |- サンプルシナリオ / ...
 |- SimRpgCon.html
```
